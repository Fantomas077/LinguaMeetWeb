
## Projekt-Schwerpunkte
- **Clean Architecture**: klare Trennung von **Business-Logik**, **Infrastruktur** und **Präsentation**  
- **Clean Code** Prinzipien für lesbaren und wartbaren Code  
- **Repository Pattern** & **Service Layer** für Datenzugriff und Geschäftslogik  
- **Unit Tests** zur Sicherstellung von Qualität und Funktionalität

## Technologien
- ASP.NET Core MVC  
- Entity Framework Core
- NUnit
- Moq
- SQL Server / SQLite  
- HTML, CSS, JavaScript , Bootstrap 5
